        Cahier de charge: Plateforme de Recrutement

1. Présentation du projet :
    JobConnect est une plateforme web de recrutement qui permet de connecter les
    entreprises avec les candidats.

    Elle permet de publier des offres d’emploi, de postuler en ligne et de suivre les
    candidatures de manière simple et organisée.

    L’objectif est de rendre le processus de recrutement plus rapide, clair et efficace.

2. Finalités du projet :
    Les objectifs principaux sont les suivants :
        ● Offrir une plateforme intuitive pour la publication et la consultation des
        offres d’emploi
        ● Permettre aux candidats de postuler rapidement en ligne
        ● Améliorer le suivi des candidatures pour les entreprises
        ● Mettre en place un environnement structuré simulant un système
        professionnel
        ● Réduire les tâches manuelles liées au recrutement

3. Types d’utilisateurs :
    La plateforme distingue trois profils d’utilisateurs :
        👤 Candidat :
            ● Création d’un compte personnel
            ● Consultation des offres disponibles
            ● Soumission de candidatures
            ● Ajout de CV
            ● Suivi de l’évolution des demandes
        🏢 Recruteur / Entreprise :
            ● Création d’un espace entreprise
            ● Publication d’annonces d’emploi
            ● Définition des critères (compétences, salaire, postes
            disponibles…)
            ● Consultation des profils candidats● Gestion des candidatures (validation ou refus)
            ● Organisation des entretiens
        🛠 Administrateur :
            ● Contrôle global de la plateforme
            ● Validation préalable des offres publiées
            ● Gestion des comptes utilisateurs
            ● Suivi des activités et des statistiques
            ● Gestion d’un système de commission interne

4. Fonctionnalités proposées :
    🔐 Système d’authentification
        ● Inscription et connexion sécurisées
        ● Attribution de rôles selon le type d’utilisateur
    📢 Gestion des offres d’emploi
        ● Création et modification d’offres
        ● Suppression si nécessaire
        ● Validation obligatoire avant publication
    📩 Gestion des candidatures
        ● Envoi de candidatures en ligne
        ● Suivi de l’état :
            ○ En attente
            ○ Validée
            ○ Refusée
    📆 Organisation des entretiens
        ● Planification des rendez-vous
        ● Consultation des dates par les candidats
    🔔 Système de notification
        ● Information des utilisateurs lors des changements de statut
        ● Alertes concernant les entretiens
    ⚙️ Fonctionnalités avancées
        ● Évaluation de compatibilité candidat/offre (matching)
        ● Ajout d’offres en favoris
        ● Gestion d’un modèle de commission pour l’administrateur

5. Environnement technique :
        Frontend :
            ● HTML
            ● CSS
            ● JavaScript
        Backend :
            ● PHP
        Accès aux données :
            ● PDO pour une connexion sécurisée
        SGBD :
            ● MySQL

6. Structure des données :
    Le système repose sur plusieurs entités principales :
        ● Utilisateurs
        ● Entreprises
        ● Offres d’emploi
        ● Candidatures
        ● Entretiens
        ● Notifications
        ● Commissions

7. Interfaces prévues :
    L’application comprendra plusieurs interfaces :
        ● Page d’accueil publique
        ● Interface candidat
        ● Interface entreprise
        ● Interface administrateur
        ● Pages de gestion (offres, candidatures…)
        
8. Contraintes du projet :
        - Application accessible via navigateur web
        - Interface ergonomique et responsive
        - Sécurité des données utilisateurs
        - Bonne gestion des accès selon les rôles
        - Optimisation des performances

9. Résultat attendu :
    Au terme de ce projet, la plateforme devra permettre :
        ● Une gestion centralisée et efficace du recrutement
        ● Une meilleure visibilité des offres d’emploi
        ● Une interaction fluide entre candidats et recruteurs
        ● Une expérience utilisateur proche des plateformes professionnelles
